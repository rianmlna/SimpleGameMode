# SimpleGameMode
**a simple gamemode plugin pocketmine**
### Commands:
```/gma```
```/gmc```
```/gms```
```/gmspc```
### Permission:
```
permissions:
  gms.cmd:
    default: op
  gmc.cmd:
    default: op
  gma.cmd:
    default: op
  gmspc.cmd:
    default: op
```
### Example command use:
```/gma```
```/gma Steve```
### Api:
**5.0.0 +**
